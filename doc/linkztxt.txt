INTRODUCTION                                    *linkztxt*

This plugin's main purpose is to make browsing a linkz.txt file more pleasant.


COMMANDS                                        *linkztxt-commands*

linkztxt					      *:LinkzList*

        Opens a list of all links defined by the linkz.txt file.

MAPPINGS                                        *linkztxt-mappings*

There are no mappings registered by the plugin.
You need to map it according to your preference.
